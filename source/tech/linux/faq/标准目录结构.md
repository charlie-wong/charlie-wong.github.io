# Linux FHS(Filesystem Hierarchy Standard)

- https://wiki.linuxfoundation.org/lsb/start
- https://refspecs.linuxfoundation.org/FHS_3.0/fhs-3.0.html
- https://linuxhandbook.com/linux-directory-structure
- https://www.geeksforgeeks.org/linux-directory-structure

- https://linuxhandbook.com/linux-memory-usage
  * 命令 `free`, `top`, `htop`, `vmstat`

- https://linuxhandbook.com/check-cpu-info-linux
  * 命令 `lscpu`, `lsmem`, `lshw`, `dmidecode`
  * 命令 `hwinfo`, `sudo apt install hwinfo`

```bash
# How to Check Linux Login History
# https://linuxhandbook.com/linux-login-history
last # View history of all successfully logged users
last UserName # View login history for a certain user

# /var/log/btmp   Logs of bad login attempts
# /var/log/wtmp   Logs of last login sessions
# /var/run/utmp   Logs of current login sessions
lastb # View all the bad login attempts
last -f /var/log/btmp
```
